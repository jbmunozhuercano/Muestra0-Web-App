Muestra0 Web App

Proyecto creado para el curso de Experto en Diseño Web de COCO School.

Descripción

Mediante la instalación del plugin de WordPress JSON API en el blog Muestra0, obtenemos y mostramos el extracto, imágenes y detalles de las últimas seis publicaciones mediante Ajax.

La finalidad de este aplicación es meramente demostrativa.

Tecnologías usadas

jQuery Mobile jQuery JSON API (Instalado en el blog)
